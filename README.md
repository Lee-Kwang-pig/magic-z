# 项目简介：
该项目将通过 Vue框架 搭建一个简洁优雅的个人网站！

主要功能：学习区（开发中）、影音区（待开发）、资料下载区（待开发）。

## 快速启动
1. 克隆到本地之后，进入根目录，安装相关依赖
```
npm install
```
2. 运行本地服务器
```
npm run serve
```
